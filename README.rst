========
Adminmax
========

Adminmax is a modern CSS Skin for the Django Admin site. 

How to install Adminmax
-----------------------

Add the path to the adminmax directory in the STATICFILES_DIRS directive.::
					
	STATICFILES_DIRS = (
     # ...
     "/path/to/adminmax/static/",
    )
